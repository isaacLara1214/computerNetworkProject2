Networking Concepts

DNS

DNS translates a human-readable domain name into an IP address so browsers can locate the correct web server.

IP Addressing

Devices on the internet communicate using IP addresses. This site can resolve to both IPv4 and IPv6 addresses.

Hosting

This website is hosted publicly so users can access it from a web browser anywhere with an internet connection.

HTTP vs HTTPS

HTTP

HTTP is the protocol used to transfer web content between clients and servers.

  • Transfers website data
  • Uses request/response model
  • Can return status codes like 200 OK

HTTPS

HTTPS is the secure version of HTTP. It encrypts communication using TLS to protect data between the client and server.

  • Encrypts traffic
  • Improves security and trust
  • Required for modern public websites
